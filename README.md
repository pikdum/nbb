# nbb

A cross-platform booru browser, written in HTML/CSS/JS and built with [http://electron.atom.io/](Electron).

Releases at: [https://files.kuudere.moe/nbb/](https://files.kuudere.moe/nbb/)

## supported boorus

* danbooru
* safebooru
* gelbooru
* yande.re

## planned enhancements
